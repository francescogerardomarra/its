# Initialize Minikube

Open a terminal and run this command:

```commandline
minikube start
```

**The previous command:**

- starts a local Kubernetes cluster using Minikube;
- provisions a virtual machine or container runtime to run the cluster;
- configures `kubectl` to use the Minikube cluster;


- downloads necessary Kubernetes components if they are not already available;
- supports additional flags to customize CPU, memory, Kubernetes version, and drivers.
