# Definition
In standard Java, dependency injection (DI)
is a design pattern used to reduce tight coupling between components
by allowing [dependencies](../dependencies/dependencies.md) to be injected, rather than hard-coded.

There are several ways to implement DI in standard Java (without frameworks like Spring).

**Here are the most common approaches:**
- [constructor](../types/constructor/example/example.md);
- [setter](../types/method/setter/example/example.md);
- [interface](../types/method/example/example.md).
